Fake News Detector API

Welcome to the Fake News Detector API. This API provides endpoints for detecting fake news and calculating trust scores for news articles.

API Documentation ReDoc Documentation API Info

Available Endpoints

POST
/predict

Predict whether a news article is real or fake.

Request Body

{
  "text": "Your news article content here"
}

Response

{
  "prediction": "REAL",
  "confidence": 0.85,
  "text_length": 1024
}
POST
/trust-score

Calculate a trust score for a news article.

Request Body

{
  "text": "Your news article content here"
}

Response

{
  "score": 75,
  "trust_level": "High Trust",
  "prediction": "REAL",
  "factors": {
    "source_credibility": 70,
    "content_analysis": 80,
    "language_analysis": 65,
    "fact_verification": 85
  },
  "details": {
    "word_count": 250,
    "has_citations": 1,
    "sensationalism_level": 0,
    "readability_score": 65.5,
    "sentiment_polarity": 0.12,
    "prediction_confidence": 0.85
  }
}
POST
/extract-url

Extract content from a URL.

Request Body

{
  "url": "https://example.com/news-article"
}

Response

{
  "title": "Example News Article",
  "content": "The extracted content of the article...",
  "source": "example.com",
  "source_credibility": 65,
  "extraction_method": "newspaper"
}
POST
/analyze-url

Analyze a news article from a URL.

Request Body

{
  "url": "https://example.com/news-article"
}

Response

{
  "prediction": "REAL",
  "confidence": 0.85,
  "trust_score": 75,
  "trust_level": "High Trust",
  "factors": {
    "source_credibility": 70,
    "content_analysis": 80,
    "language_analysis": 65,
    "fact_verification": 85
  },
  "article": {
    "title": "Example News Article",
    "content": "The extracted content of the article...",
    "source": "example.com",
    "source_credibility": 65,
    "extraction_method": "newspaper",
    "url": "https://example.com/news-article"
  }
}

Example Usage

Here's an example of how to use the API with curl:

curl -X POST "http://localhost:8000/predict" \
     -H "Content-Type: application/json" \
     -d '{"text": "Scientists at MIT have developed a new type of battery that can store renewable energy for months without losing charge."}'

Example of analyzing a URL:

curl -X POST "http://localhost:8000/analyze-url" \
     -H "Content-Type: application/json" \
     -d '{"url": "https://www.bbc.com/news/science-environment-56837908"}'
